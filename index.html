Miscommunications

i need RUBY or Node on backend

  • Sass is written in ruby
  • LESS is written in javascript
  • This does NOT required Ruby or Node on your prod env
  • You need them on you development machine

LESS runs client-side

  • LESS can run client-side
  • OK for testing and if you can't access dev env
  • NEVER EVER use it in prod. it's even worce than @import performance
  • Use Node. of Rhino

IT's hard to install

  • Both ruby and Node are easy to install on any platform
  • Mac: Ruby from the box, Node.pkg
  • Windows: RubyInstaller.exe, Node.exe
  • unix: apt-get install ruby/nodejs
